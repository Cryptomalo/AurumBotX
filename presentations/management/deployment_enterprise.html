Docker & Compose

Containerizzazione completa con Dockerfile ottimizzato e docker-compose.yml per gestione semplificata del servizio con persistenza dei dati (volumi per data, logs, reports).

Script VPS

Script deploy_vps.sh che automatizza l'installazione di Docker, clonazione repository, configurazione variabili d'ambiente e avvio container su server remoti Linux.

Pipeline CI/CD

GitHub Actions workflow per testing automatico (flake8, pytest), build Docker e push su Docker Hub. Notifiche Telegram al completamento di ogni deployment.

Packaging Stand-Alone

PyInstaller con file spec per creare eseguibili nativi per Windows, macOS e Linux. Inclusi tutti i file di risorse (PWA, configurazioni, analytics).
